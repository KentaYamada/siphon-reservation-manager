# Siphon reservation manager
## 概要
小規模店舗向けの予約受付アプリケーションです。  
座席指定の予約をすることが可能です。

### 機能
- 予約
  - 受付
  - 内容確認
  - 変更
  - キャンセル
- 設定
  - 営業日
  - 予約時間帯

### 技術要素
- Frontend

  - [Buefy](https://buefy.org/)

  - [firebase](https://firebase.google.com/?hl=ja)

  - [jest](https://jestjs.io/ja/)
